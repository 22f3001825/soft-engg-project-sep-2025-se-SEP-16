# Frontend

Intellica is a customer service platform. It provides separate portals for Customers, Agents, Supervisors, and Vendors, allowing efficient order and support management.

## Project Setup

### Install Dependencies

```bash
cd frontend
yarn install
```

### Compile and Hot-Reload for Development

```bash
yarn start
```

### Compile and Minify for Production

```bash
yarn build
```
