# Home & Garden Category – Return Policy

## Eligibility
- Returns allowed within **14 days** of delivery.
- Product must be **unused**, with original packaging and tags.
- All components (for sets) must be included.

## Non-Eligible Items
- Customized/personalized items (wall art with custom text, engraved mugs).
- Used candles, diffusers, or consumables (hygiene/safety).
- Plants (living items; covered under separate plant guarantee policy).
- Assembled furniture (if customer assembled; disassembly not accepted).

## Defect/Damage Returns
- **Chipped/broken on arrival**: Photos required (multiple angles).
- **Manufacturing defect** (e.g., uneven pottery, faulty lamp wiring): Vendor QC verification required.
- **Color/design mismatch**: Product listing screenshot + received item photo for comparison.

## Quality Check Process
1. Customer initiates return.
2. Product inspected at warehouse for usage signs.
3. If unused → Full refund.
4. If used (stains, wear, missing tags) → Refund denied.

## Partial Refunds
- **Missing components (from set)**: Deduct component value.
- **Minor cosmetic damage (customer-caused)**: 30% deduction.
- **Opened consumables (oils, candles) but unused**: 15% restocking fee.

## Exception Handling
- Supervisor can approve refund for customized items if:
  - Production error (wrong text, image quality issue).
  - Damaged during shipping (requires photo proof).
