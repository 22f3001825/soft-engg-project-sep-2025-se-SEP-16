# Refund Eligibility Guide for Customers

## Quick Eligibility Checker

### Am I Eligible for a Refund?

**YES - Full Refund** if:
- Item is defective or doesn't work
- Item arrived damaged
- Wrong item was shipped
- Item significantly different from description
- Return within 30 days (14 days for electronics)
- Item in original condition with packaging

**MAYBE - Partial Refund** if:
- Missing original packaging (15% fee)
- Missing accessories (cost deducted)
- Item shows minor wear (up to 30% deduction)
- Return between 31-45 days (20% fee)

**NO - Not Eligible** if:
- Return after 45 days
- Item is non-returnable (food, personal care, custom items)
- Item damaged by customer
- No proof of purchase
- Item not in resellable condition

## Eligibility by Product Category

### Electronics & Technology (14-day window)
**Eligible:**
- Unopened items in original packaging
- Defective items (any condition)
- Wrong item shipped

**Not Eligible:**
- Opened software or digital products
- Items with missing serial numbers
- Damaged by customer misuse

**Special Notes:**
- Must include all cables, accessories, manuals
- Original packaging required
- Restocking fee may apply if opened

### Apparel & Fashion (30-day window)
**Eligible:**
- Unworn items with tags attached
- Wrong size or color
- Defective items (tears, stains, manufacturing defects)

**Not Eligible:**
- Worn or washed items
- Items without tags
- Altered or tailored items
- Intimate apparel (if opened)

**Special Notes:**
- Must be in original condition
- No signs of wear, perfume, or deodorant marks
- Shoes must have unworn soles

### Home & Furniture (30-day window)
**Eligible:**
- Unused items in original packaging
- Damaged during shipping
- Assembly defects

**Not Eligible:**
- Assembled furniture (unless defective)
- Custom or made-to-order items
- Items showing signs of use

**Special Notes:**
- Large items may require special return arrangements
- Assembly instructions must be included
- All hardware and parts must be present

### Food & Beverages (7-day window)
**Eligible:**
- Unopened, sealed items
- Items past expiration date upon arrival
- Damaged or leaking packages

**Not Eligible:**
- Opened food items
- Perishable goods past 7 days
- Items stored improperly

**Special Notes:**
- Must be unopened and sealed
- Temperature-sensitive items have 48-hour window
- Photos required for damaged items

### Health & Beauty (30-day window)
**Eligible:**
- Unopened items in original packaging
- Sealed products with safety seal intact
- Defective products

**Not Eligible:**
- Opened cosmetics or skincare
- Used personal care items
- Items without safety seals

**Special Notes:**
- Hygiene and safety regulations apply
- Must be completely unused
- Original packaging required

## Refund Eligibility by Reason

### Defective or Damaged Items
**Eligibility**: 100% - Full refund + free return shipping

**Requirements**:
- Report within 48 hours of delivery
- Provide photos of defect/damage
- Include photos of packaging
- Item doesn't need to be in original condition

**Process**:
- Immediate approval
- No inspection delay
- Refund processed upon receipt
- Option for replacement instead

### Wrong Item Shipped
**Eligibility**: 100% - Full refund + free return shipping

**Requirements**:
- Report immediately upon receipt
- Provide photo of item received
- Keep item until replacement arrives

**Process**:
- Correct item shipped immediately
- Free expedited shipping
- Return label provided
- No refund delay

### Change of Mind / No Longer Needed
**Eligibility**: Full refund if within return window

**Requirements**:
- Return within category-specific window
- Item in original, unused condition
- All packaging, tags, and accessories included
- Customer pays return shipping ($6.99)

**Process**:
- Standard inspection (2-3 days)
- Return shipping cost deducted from refund
- Refund processed after approval

### Item Not as Described
**Eligibility**: 100% - Full refund + free return shipping

**Requirements**:
- Significant difference from product description
- Provide photos showing discrepancy
- Report within 7 days of delivery

**Process**:
- Support team reviews claim
- Free return shipping if claim approved
- Full refund upon receipt

### Found Better Price
**Eligibility**: Full refund minus return shipping

**Requirements**:
- Return within standard window
- Item in original condition
- Customer pays return shipping

**Alternative**:
- We may offer price match instead of return
- Contact support with competitor's price

### Size or Fit Issues
**Eligibility**: Full refund or free exchange

**Requirements**:
- Item unworn with tags attached
- Return within 30 days
- Original packaging preferred

**Process**:
- Free exchange shipping (both ways)
- Or standard return process
- Size guides available to prevent future issues

## Special Eligibility Situations

### Gift Returns
**Eligible if**:
- Gift receipt provided
- Within return window
- Item meets condition requirements

**Refund Options**:
- Store credit (no gift receipt needed)
- Refund to gift giver's payment method (with gift receipt)
- Exchange for different item

### Sale or Clearance Items
**Eligible if**:
- Not marked "Final Sale"
- Within return window (may be shorter - 14 days)
- All standard conditions apply

**Not Eligible**:
- Items marked "Final Sale" or "As-Is"
- Clearance items over 75% off (store policy)

### Promotional or Free Items
**Eligible if**:
- Part of defective order
- Wrong item shipped

**Not Eligible**:
- Free gifts with purchase (unless defective)
- Promotional samples

### Subscription or Auto-Ship Items
**Eligible if**:
- Cancel subscription before next shipment
- Return current shipment within window
- Standard return conditions apply

**Process**:
- Cancel subscription first
- Then initiate return
- Refund for returned items only

## Tips for Successful Returns

**To Ensure Full Refund**:
1. Return within the specified window
2. Keep all original packaging
3. Don't remove tags or labels
4. Include all accessories and manuals
5. Pack securely to prevent damage in transit
6. Get tracking number and keep receipt

**To Speed Up Process**:
1. Initiate return online (faster than phone/email)
2. Provide clear photos if item is defective
3. Include order number in package
4. Use provided return label
5. Ship within 5 days of initiating return

**To Avoid Issues**:
1. Read product description carefully before ordering
2. Check size guides for apparel
3. Review return policy before purchasing
4. Keep order confirmation email
5. Inspect items immediately upon delivery
6. Report issues within 48 hours
